## How to Save Images:

1. Right-click on each image attachment
2. Save it with the corresponding name above
3. Make sure all images are in **JPG** format
4. Place all images in this **images** folder

✅ بعد حفظ الصور، افتح **index.html** في المتصفح لرؤية المنتجات بالصور الحقيقية!

✅ After saving images, open **index.html** in browser to see products with real images!
